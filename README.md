# GroupNavi ROS2

This repository is adapted from [GenSafeNav-ROS2](https://github.com/tasl-lab/GenSafeNav-ROS2), the ROS2 deployment codebase for the paper _[Towards Generalizable Safety in Crowd Navigation via Conformal Uncertainty Handling](https://arxiv.org/abs/2508.05634v1)_.

For setup instructions, component overview, and citation details, please refer to the [original repository](https://github.com/tasl-lab/GenSafeNav-ROS2).
